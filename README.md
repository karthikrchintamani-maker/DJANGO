# Django Student Management Project

A simple web application built using the **Django Framework**. This project demonstrates the basic concepts of Django, including URL routing, views, templates, static content, and project organization. It serves as a beginner-friendly project for understanding how Django applications are developed and managed.

---

# What is Django?

Django is a **high-level Python web framework** that enables rapid development of secure, scalable, and maintainable web applications. It follows the **Model-View-Template (MVT)** architecture, which separates application logic, user interface, and data management.

Django was designed to help developers build web applications quickly while avoiding repetitive coding. It comes with many built-in features such as authentication, URL routing, ORM (Object Relational Mapper), form handling, session management, security protections, and an administrative interface.

# How Django Works

1. User sends a request through the browser.
2. Django receives the request.
3. URL Dispatcher matches the URL.
4. View function processes the request.
5. Data is sent to the template.
6. Template generates HTML.
7. HTML response is returned to the browser.

# Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project directory

```bash
cd django_project
```

Install Django

```bash
pip install django
```

Apply migrations

```bash
python manage.py migrate
```

Run the server

```bash
python manage.py runserver
```

Open the browser

```
http://127.0.0.1:8000/
```
